2. Datenerhebung und -verarbeitung

MillyLog speichert alle eingegebenen Daten (Stillzeiten, Mahlzeiten, Notizen) ausschließlich lokal auf Ihrem Gerät. Es werden keine Daten an externe Server übertragen.

3. Gespeicherte Daten

  • Zeitstempel von Still- und Fütterungszeiten
  • Notizen zu Mahlzeiten
  • Babyprofile (Name, Geburtsdatum, Geschlecht, Profilbild)
  • Exportierte JSON/CSV-Dateien (nur bei manuellem Export durch Sie)

4. Datenweitergabe

Es erfolgt keine Weitergabe von Daten an Dritte. Die App enthält keine Analyse-Tools, Werbung oder Tracking-Mechanismen.

5. Ihre Rechte

Sie können jederzeit alle Daten löschen, indem Sie die App deinstallieren. Beim Export haben Sie volle Kontrolle über Ihre Daten. Gemäß DSGVO haben Sie das Recht auf Auskunft, Berichtigung und Löschung Ihrer Daten.
